Welcome to Puzzle Pazza Page.

Puzzla Pazza is a simple app created for demonstrating HTML5 features (built in 2011) based on canvas element.

Try it yourself

You can find and play Puzzle Pazza on puzzle.pumaja.com.

Unfortunately, the game doesn`t support any score table. The purpose of this app was to test HTML5 canvas element and keep it simple as possible.
